# Prerequisites

If you want to **successfully block** releases and never have them pop back up in your queue, you need to do the following:

<ElementNavigator />

<div className={styles.documentationPage}>

<div className={styles.section}>

<SectionTitle>Configure Prowlarr</SectionTitle>

<div className={styles.sideBySide}>
  <div className={styles.sideBySideContent}>

If you are using Prowlarr as your indexer manager:
1. Go to Prowlarr
2. Navigate to `Settings`
3. Navigate to `Apps`
4. Enabled advanced settings by clicking on `Show Advanced`
5. Edit app
6. Enable `Sync Reject Blocklisted Torrent Hashes While Grabbing`

Repeat steps #5 and #6 for each Prowlarr app you have configured.

  </div>
  <div className={styles.sideBySideImage}>
    <ClickableImage
      src={useBaseUrl('/img/prowlarr_settings.png')}
      alt="Prowlarr settings showing Sync Reject Blocklisted Torrent Hashes option"
      style={{ borderRadius: '8px', boxShadow: '0 2px 8px rgba(0,0,0,0.1)', maxWidth: '100%' }}
    />
  </div>
</div>

</div>

<div className={styles.section}>

<SectionTitle>Configure *arr</SectionTitle>

<div className={styles.sideBySide}>
  <div className={styles.sideBySideContent}>

<Important>
If you are using Prowlarr, this step is not necessary.
</Important>

If you are using Sonarr/Radarr/Lidarr/Readarr/Whisparr as your indexer manager:
1. Go to Sonarr/Radarr/Lidarr/Readarr/Whisparr
2. Navigate to `Settings`
3. Navigate to `Indexers`
4. Enabled advanced settings by clicking on `Show Advanced`
5. Edit indexer
6. Enable `Reject Blocklisted Torrent Hashes While Grabbing`

Repeat steps #5 and #6 for each indexer you have configured.

  </div>
  <div className={styles.sideBySideImage}>
    <ClickableImage
      src={useBaseUrl('/img/sonarr_settings.png')}
      alt="Sonarr settings showing Reject Blocklisted Torrent Hashes option"
      style={{ borderRadius: '8px', boxShadow: '0 2px 8px rgba(0,0,0,0.1)', maxWidth: '100%' }}
    />
  </div>
</div>

</div>

</div>
